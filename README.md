# MAMAA Technical Interview Solutions

This repository tracks my journey solving high-frequency coding problems asked by **Meta, Amazon, Microsoft, Apple, and Alphabet (Google)**. All problems are sourced from the **TutorialsPoint MAMAA Problem Sets**.

## The Goal
To master Data Structures and Algorithms (DSA) by solving real-world interview questions with a focus on:
* **Time Complexity:** Aiming for the most optimal Big O.
* **Code Clarity:** Writing "Interview-Ready" code that is easy to explain.
* **Consistency:** Daily updates as I progress through the TutorialsPoint list.

## Repository Roadmap
Since I am following the TutorialsPoint list, the solutions are organized by their company focus:

* `/Meta` - Focus on Arrays, Strings, and Binary Trees.
* `/Amazon` - Focus on Dynamic Programming and Hashing.
* `/Google` - Focus on Graphs and Complex Algorithms.
* `/Microsoft` - Focus on Linked Lists and System Logic.

## Tech Stack
* **Language:** Python 3.10+
* **Input Parsing:** Standard `sys.stdin` and `json` for competitive programming environments.
